LuckPerms
LuckPerms permite gestionar permisos de comandos / herramientas o mecanicas a grupos / jugadores especificos
Comandos
Nombre
ApplyEdits2
Funcion
Aplica Cambios en los permisos configurados desde el editor web
Sintaxis
/lp applyedits [code]
Parametros
Code
código proporcionado por el editor web tras realizar los cambios
Nombre
bulkupdate
Funcion
Permite Manipular grandes cantidades de permisos con una unica consulta sql
Sintaxis
/lp bulkupdate [data type] [action] [action field] [action value] [constraint]
Parametros
Data Type
El tipo de datos que se están modificando. (Puede ser "todos", "usuarios" o "grupos")
Action
La acción a realizar sobre los datos. (Puede ser "actualizar" o "eliminar")
Action Field
El campo sobre el que se actuará. Solo es necesario para las acciones de actualización. (Puede ser "permiso", "servidor" o "mundo")
Action Value
El valor que se reemplazará. Solo se requiere para las acciones de actualización.
Constraint
Las restricciones necesarias para la actualización
